# Ejercicio: Uso de un Reusable Workflow desde otro repositorio

## Objetivo

Aprender a utilizar un **Reusable Workflow** que se encuentra en **otro
repositorio**, referenciándolo correctamente mediante la sintaxis
`OWNER/REPO` en GitHub Actions.

------------------------------------------------------------------------

## Tareas

### 1. Craar un repo y mover allí el archivo 16-1-reusable-workflow.yaml

Crea el repositorio en git y mete el archivo en
/.github/workflows



### 1. Modificar el workflow que llama al reusable workflow

Abrir el archivo:

    .github/workflows/16-2-reusable-workflow.yaml

Buscar la línea:

    uses: ./.github/workflows/16-1-reusable-workflow.yaml

y reemplazarla por:

    uses: <OWNER>/<REPO>/.github/workflows/16-1-reusable-workflow.yaml@main

------------------------------------------------------------------------

### 2. Ejemplo

Ejemplo de uso con un repositorio real:

    uses: my-org/github-actions-reusable-workflows/.github/workflows/16-1-reusable-workflow.yaml@main

Donde:

-   **OWNER** → organización o usuario propietario del repositorio.
-   **REPO** → nombre del repositorio donde se encuentra el reusable
    workflow.
-   **@main** → rama o tag desde donde se ejecutará el workflow.

------------------------------------------------------------------------

## Verificación

1.  Hacer **commit** de todos los cambios.
2.  Hacer **push** al repositorio.
3.  Ir a la pestaña **Actions** en GitHub.
4.  Ejecutar el workflow manualmente.
5.  Verificar que:

-   El **workflow reutilizable se ejecuta correctamente**.
-   Los **outputs del workflow se imprimen en los logs**.
